You can setting tethring.

ROOT化せずに、テザリングが可能です。

シンプルにテザリングのON/OFFをするだけでなく、SSID/パスワードを変更可能！
WPA/WPA2による暗号化を行えます。
パスワードを空に設定すると、認証無しの接続となります。

設定画面が無効化されているAQUOS PHONE 104SHでも設定可能！

104SHでしか動作確認を行なっていないので、
他端末での動作確認や、不具合があれば、レビューコメントをお願いします。
